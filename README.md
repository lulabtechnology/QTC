# Quality Techno Services Landing

Landing page premium one-page construida con:

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- lucide-react
- Route Handler para formulario compatible con Vercel

## Variables de entorno

Crea un archivo `.env.local` en desarrollo o configura estas variables en Vercel:

- `RESEND_API_KEY`
- `CONTACT_TO_EMAIL`
- `CONTACT_FROM_EMAIL`
- `WHATSAPP_NUMBER`
- `LINKEDIN_URL`
- `SITE_URL`

## Comandos

```bash
npm install
npm run dev
npm run build
npm run start
```

## Assets editables

Todas las imágenes principales están preparadas en `/public` con rutas separadas desktop/mobile.
